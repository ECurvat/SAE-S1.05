Les matières

Pour programmer

C

Abordé en entièreté durant le premier semestre, le langage de programmation C est impératif, les instructions sont exécutées pour transformer l’état actuel du programme. Il utilise des variables, tableaux, pointeurs, types...

SQL

Le SQL est vu pendant les deux premiers semestres. C’est un langage de programmation utilisé pour manipuler des bases de données : création, modification, récupération de données, ajout de données...

HTML

HTML est un langage utilisé pour créer la structure des sites internet. C’est un langage utilisant des balises pour annoter du texte, des images et d’autres contenus avec comme finalité de les afficher dans un navigateur web.

CSS

CSS vient compléter HTML pour la forme d’une page web. CSS permet d’appliquer des règles de style sur des éléments sélectionnés avec des sélecteurs. Ce langage décrit comment chaque élément HTML doit être affiché.

JAVA

Ce cours présente l’historique du langage, la compilation et l’interprétation, son utilisation et surtout les bases avec les package, identificateurs, opérateurs, variables...

PHP

Présentation du langage PHP, comment il fonctionne, la syntaxe, comment l’intégrer à une page web existante, la transmission de données via un formulaire, la connexion à MySQL…

JAVASCRIPT

JavaScript est un langage à scripts qui permet d’implémenter des mécanismes plus complexes sur une page web. Il permet entre autres d’inclure des animations complexes, interactions. En général, on peut créer du contenu mis à jour de façon dynamique.

Pour comprendre l'informatique

Mathématiques

Les bases des mathématiques sont revues en début de formation pour ensuite aborder des notions plus complexes qui se rapprochent de plus en plus de l’informatique.

Algorithmie

L’algorithmie est une matière étroitement liée avec la programmation en C. Apprentissage des boucles, types, fonctions. Le but principal de cette matière est de trouver l’algorithme qui sera le plus léger, le moins complexe possible et qui répond évidemment au problème posé.

Architecture

Dans cette matière, différents thèmes sont abordés. Un peu d’histoire de l’informatique, les composants d’un ordinateur, les circuits électroniques, les portes logiques...

Réseau

Le cours de réseau présente les protocoles, ce qu’est un port, les serveurs, les DNS. Cette matière est très utile pour des étudiants souhaitant se spécialiser dans le parcours déploiement d’applications communicantes et sécurisées.

Systèmes d'exploitation

Présentation des systèmes d’exploitations, leur histoire, comment bien les utiliser. Le cours de système se fait avec des machines virtuelles et les étudiants travaillent en lignes de commandes principalement.

Base de donnée

Dans le cours de conception de bases de données, l’étudiant apprend à créer un modèle à partir de contraintes clients spécifiques. Il comprend aussi pourquoi les bases de données sont nécessaires, les principes et les règles à respecter.

Git

Le cours de Git est très utile et est réalisé en début de formation pour pouvoir être utilisé lors de projets à rendre. Il présente l’avantage des systèmes de contrôle des versions et comment les utiliser.

Pour travailler en entreprise

Communication

Le cours de communication est très général mais permet d’apprendre des outils dont l’étudiant diplômé se servira tous les jours. Il offre une première approche de ce qui est attendu en entreprise et le comportement à adopter.

Projet Personnel et Professionnel

Ce cours est centré sur la découverte de soi, de ses buts, ses valeurs, ses centres d’intérêts, les domaines qui nous intéressent, les parcours professionnels envisagés...

Anglais

Très utilisée en entreprise, l’anglais est de nos jours un réel plus sur un CV si maîtrisé. Ce sont principalement des projets à réaliser, des présentations à faire, qui reste cependant toujours en rapport avec l’informatique.

Droit

Le cours de droit s’applique directement à l’informatique et permet à l’étudiant de se rendre compte de l’organisation du droit en France et à l’international. Une grande partie de ce cours est consacrée aux contrats.

Economie

C’est un cours sur l’économie en général. Il aide beaucoup à comprendre le fonctionnement de la société actuelle. Sont vus le rôle de l'État, les politiques économiques, la mondialisation et les enjeux du développement durable.

Gestion des organisations

Le cours de gestion des organisations est la suite directe du cours d’économie. Le but est de comprendre comment une organisation fonctionne, ses choix, les types d’organisations… Ce cours offre une première approche théorique à la vie professionnelle.
